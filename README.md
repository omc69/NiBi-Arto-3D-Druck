# NiBi-Arto-3D-Druck
Parametrische 3D-Modelle erstellt in OpenSCAD.
Alle Modelle lassen sich durch einfaches Anpassen der Parameter an eigene Maße anpassen und direkt als STL für den 3D-Druck exportieren.
